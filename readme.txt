【基本的な機能】
・ニコニコ生放送(http://live.nicovideo.jp/)において、
  あらかじめ登録したキーに合う生放送が開始された場合に、
  音を鳴らす/バルーン表示する/ブラウザを起動するなどによって通知するツールです。
  登録キーは、コミュニティ番号、チャンネル番号、ユーザID、放送番号が登録でき、
  登録毎に通知方法を設定することができます。

・Love は 0 って意味ですよ？
  ソフト名称は、生放送中に頂いた案の中から、
  ZKevTWRsqmIOITcfzfyhhakouAM様の案を採用させて頂きました。

【注意事項】
・本ツールはニコニコ生放送公式ツールではありません。
  絶対に公式への問い合わせはしないでください。

・ログインセッションを必要としないことをコンセプトとしているため、
  ユーザー名の自動取得については、取得手段が無いため対応できません。

・チャンネル番号を登録する場合、該当チャンネルにて生放送が行われているかどうか
  確認するようお願いします。
  チャンネル生放送のように見えても実際は公式生放送として放送される場合があるため、
  この場合、ch登録では検出できず、放送番号(lv)登録が必要となります。
  ⇒アラートAPIの仕様です。
    チャンネルページを開いて放送履歴が無い場合、
    そのチャンネルでは生放送が行われていません。

・カスタマイズツール等を用い、レジストリでバルーン表示を無効化している場合は、
  バルーン表示ができません。
  (窓の手、Tweak UI等で無効化している場合はご注意ください。)

===============================================================================
【インストール方法】
・適当なフォルダに、ZIPファイル内の全てのファイル展開するだけです。
  Windows Vista以上の場合、Program FilesまたはProgram Files (x86)配下に入れると、
  データベースへの書き込みエラーが発生する場合があります。

===============================================================================
【アンインストール方法】
・インストールしたフォルダ内のファイルを削除してください。
  レジストリは未使用です。
